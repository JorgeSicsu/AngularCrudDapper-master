✅ Passos para subir esse projeto Angular 20 no Railway como SPA:

1. Execute:
   npm install --legacy-peer-deps

2. Execute:
   npm run build

3. O comando 'npm start' irá rodar o projeto com http-server usando o diretório dist/angular-dapper-crud

O deploy no Railway utilizará automaticamente o comando:
   npm start

⚠️ Este projeto NÃO usa SSR.
